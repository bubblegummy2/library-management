## Test Structure

**Suite Organization:**
- Not applicable. There are no unit test suites.

**Patterns:**
- `scripts/verify_doc_links.py` is a script-style check:
  - Iterates over `docs/*.md`.
  - Extracts markdown links using `re.finditer`.
  - Skips external URLs and anchor-only links.
  - Resolves local links relative to the source markdown file.
  - Prints each broken link and exits non-zero if any are found.
